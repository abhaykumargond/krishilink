# KrishiLink – Bridging Farmers to the Future

KrishiLink is a modern, responsive web platform designed to empower farmers with smart agriculture solutions powered by IoT, AI, and Blockchain technologies. This project provides a user-friendly interface for farmers to monitor their crops, access marketplace services, and utilize various agricultural tools.

## Features

- **Dashboard**: Real-time monitoring of soil moisture, temperature, humidity, and crop health
- **Marketplace**: Buy and sell agricultural products with ease
- **Blockchain Traceability**: Track the journey of agricultural products from farm to consumer
- **Crowdfunding**: Support farmers through secure crowdfunding campaigns
- **Tokenized Farmland**: Invest in farmland through tokenization
- **Smart Insurance**: Protect your crops with AI-powered insurance solutions

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Charts**: Chart.js for data visualization
- **Icons**: Font Awesome
- **Fonts**: Google Fonts (Poppins)
- **Responsive Design**: Mobile-first approach with CSS Grid and Flexbox

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No backend or database setup required (static website)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/krishilink.git
   ```

2. Navigate to the project directory:
   ```bash
   cd krishilink
   ```

3. Open `index.html` in your preferred web browser.

## Usage

1. Open `index.html` in a web browser to view the landing page.
2. Navigate to different sections using the navigation menu.
3. The dashboard provides interactive charts and real-time data visualization.
4. Explore various features like marketplace, traceability, and more.

## Customization

### Colors

The application uses CSS variables for easy theming. You can customize the colors by modifying the following variables in `css/style.css`:

```css
:root {
    --primary-color: #2ECC71;
    --primary-dark: #27AE60;
    --secondary-color: #0F3C68;
    --text-color: #333;
    --light-gray: #f5f7fa;
    --white: #ffffff;
    --shadow: 0 5px 15px rgba(0, 0, 0, 0.1);
    --transition: all 0.3s ease;
}
```

### Adding New Pages

1. Create a new HTML file in the `pages` directory.
2. Link the required CSS and JavaScript files.
3. Use the existing components and styles for consistency.
4. Add a link to the new page in the navigation menu.
